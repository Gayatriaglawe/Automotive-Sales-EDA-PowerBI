Automotive Data Analysis

This project is based on an automotive dataset in Excel format.
In this project, I worked on cleaning, analyzing and understanding vehicle-related data to find useful insights.

About the Dataset

The dataset contains different details about vehicles such as:

Car name

Price

Engine size

Fuel type

Mileage

Other specifications

The data was available in Excel format and required cleaning before analysis.

What I Did in This Project

Removed missing and duplicate values

Checked data types and corrected them

Performed basic statistical analysis

Compared fuel types and pricing

Analyzed relationship between engine size and price

Created charts to visualize trends

Tools Used

Microsoft Excel

Basic data analysis techniques

Charts and Pivot Tables

Key Learnings

Through this project, I improved my understanding of:

Data cleaning

Data visualization

Working with real-world datasets

Finding patterns and trends in data

Conclusion

This project helped me practice data analysis skills using a real dataset. It improved my confidence in handling Excel-based data and presenting insights clearly.
